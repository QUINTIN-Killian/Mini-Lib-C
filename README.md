# Mini-Lib-C

This project implements certain of the main functions of the C library in assembly.

---

## ðŸ§  Description

The implemented functions are the following:

- strlen
- strchr
- strrchr
- memset
- memcpy
- strcmp
- strncmp
- strcasecmp
- strpbrk
- strstr
- strcspn

They are all compiled in the libasm.so library, that an then be used in any project.

**Compiling:**

